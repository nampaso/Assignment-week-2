# Personal Budget & Expense Tracker

## Project Overview

This project is a Personal Budget & Expense Tracker built using HTML and CSS.

The purpose of the application is to help users track expenses by entering expense information and viewing expenses in a structured table.

## Features Implemented (Week 2)

### Page Header
- Budget Tracker title
- Logo image
- Short description

### Add Expense Form
- Expense Name input
- Expense Amount input
- Category dropdown
- Add Expense button

### Expense Table
Displays sample expenses using:
- table
- thead
- tbody
- tr
- th
- td

### Help Section
Uses HTML details and summary elements to provide instructions.

### Multimedia
- Logo image
- Embedded YouTube budgeting video

### CSS Features
- Element selectors
- Class selectors
- ID selectors
- Descendant selector
- Direct child selector
- Focus pseudo-class
- Negation selector
- nth-child pseudo-class
- Hover effects

## Future Development

Week 3 and later versions will add:
- JavaScript functionality
- Dynamic expense creation
